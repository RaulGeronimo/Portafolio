¿En qué consiste este CRUD?

CRUD Tienda Angular

Para la realizacion de este CRUD se utilizaron los siguientes frameworks: Angular, TypeScript y Node JS asi como el gestor de bases de datos MySQL.

Este CRUD esta realizado en base a una tienda cualquiera, en donde el usuario podra agregar productos del mismo, asi como modificarlos y finalmente eliminarlos.

Descargar Proyecto

Diagrama de Base de Datos

Este diagrama consiste de un total de 1 tabla, la cual es: Producto.

Compilado

Para poder compilar este proyecto, primero deberas contar con Visual Studio Code, asi como un gestor de bases de Datos.

Primero se debera modificar el Archivo de data que se encuantra en la carpeta de Servidor.

Ahora para poder compilar el proyecto, se deben abrir 3 terminales, en la primer terminal se accedera a la carpeta Servidor y posteriormente se colocara el comando npm run build.

En la segunda terminal se accedera a la carpeta Servidor y posteriormente se colocara el comando npm run dev.

En la ultima terminal se accedera a la carpeta appTienda y posteriormente se colocara el comando ng serve --open.

Manejo del Programa

Para poder insertar registros, se podra dirigir a la pestaña y de Agregar Producto una vez que haya terminado de llenar los campos le dara en Aceptar.

Lista

Para poder visualizar los registros insertados se dirigira a la pestaña de Productos.
