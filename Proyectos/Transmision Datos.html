¿En qué consiste este CRUD?

CRUD Transmisión de Datos

Para la realización de este CRUD se utilizaron los siguientes lenguajes: HTML, JavaScript, CSS y PHP asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a una página que sirve para ayudar a estudiantes que estén cursando esta asignatura, esta página es un material de apoyo hacia estos alumnos.

Compilado

Para poder compilar este proyecto, primero deberás contar algun editor de texto, asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de Conexion.php que se encuentra en la carpeta de PHP.


Ahora para poder compilar el proyecto, se deberán activar los servicios de apache y MySQL en Xammp.


Finalmente para ejecutar el proyecto, se debera abrir el navegador y colocar la direccion http://localhost/Transmision%20de%20Datos/. Antes de ejecutarlo se debera asegurar que el proyecto se encuentre en la carpeta C:\xampp\htdocs

Manejo del Programa

Para poder entrar a la pagina, primero se debera registrar el usuario.

Una vez insertado el usuario, para navegar a través de la página viendo el contenido de los temas que posee esta pagina
