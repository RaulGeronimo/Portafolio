¿En qué consiste este CRUD?

CRUD Notas (Aplicación completa)

Para la realización de este CRUD se utilizaron los siguientes lenguajes: Nodejs, Mongodb, Express, para el diseño se utilizaron los framework de CSS y Bootstrap asi como el gestor de bases de datos MongoDB.

Este CRUD está realizado en base a una lista de notas en donde el usuario podra escribir sus notas, asi como un login para asi separar los registros de cada usuario.
