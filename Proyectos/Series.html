¿En qué consiste este CRUD?

CRUD Series Spring

Para la realización de este CRUD se utilizaron los siguientes lenguajes: Java, JSP y Spring MVC asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a un repertorio de Series, asi como de películas, caricaturas y animes, en donde el usuario podrá agregar a las Temporadas de cada uno y también los directores encargados de cada serie, finalmente podrá modificarlos y finalmente eliminarlos.

Diagrama de Base de Datos

Este diagrama consiste en un total de 16 tablas, las cuales son: anime, caricatura, clasificacion, director, distribuidora, estudioanimacion, genero, pais, peliculas, productora, series, temporadas_anime, temporadas_caricatura, temporadas_emision, temporadas_series y usuario. Este proyecto igual cuenta con 23 vistas

Compilado

Para poder compilar este proyecto, primero deberás contar algun editor de código (de preferencia NetBeans), asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de ConexionDB que se encuentra en la carpeta de config.


Ahora para poder compilar el proyecto, se dará clic en el botón de la flecha verde o en su defecto con la tecla F6.

Manejo del Programa

Para poder empezar a utilizar el programa, primero deberá registrar un usuario para que este lo deje iniciar sesión.

Contenido del Programa

Una vez dentro del programa, aparecerá una sección de contenido del programa, en este podrá seleccionar la sección en la que desee estar.

Contenido por sección

En cada una de las secciones se encontrara un submenu en lo que podra elegir que desea insertar.

Agregar Registros

Para poder insertar registros, se podrá dirigir a cualquiera de los apartados que aparecen en el menú de cada una de las secciones.

Lista

Para poder visualizar los registros insertados se podrá dirigir al primer apartado de cada uno de los menús del programa. Asimismo, en cada una de las tablas se encontrará en la parte superior una serie de botones que servirán para exportar los registros.

Existen algunos apartados que poseen 2 tipos de vistas, en donde se puede ver la cantidad de capítulos y su año de estreno. En estos apartados se realizó una agrupación asi como el conteo de capítulos y el promedio de la serie o caricatura.
