¿En qué consiste este CRUD?

CRUD Series Spring

Para la realización de este CRUD se utilizaron los siguientes frameworks: Java, JSP y Spring MVC asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a un repertorio de Series, asi como de películas, caricaturas y animes, en donde el usuario podrá agregar a las Temporadas de cada uno y también los directores encargados de cada serie, finalmente podrá modificarlos y finalmente eliminarlos.
